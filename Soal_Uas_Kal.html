1.Hitunglah PageRank dari graph berikut

untuk menghitung pagerank tersebut dari graf yang diberikan kita terlebih dahulu cari algoritma pagerank nya dengan langkah langkah berikut :

1.inisialisasi nilai pagerank

2.terapkan rumus dari pagerank

3.hitung nilai pagerank secara iteratif

4.implementasikan pada python

#implementasi
import numpy as np


d = 0.85


adj_matrix = np.array([
    [0, 0, 1, 1, 0],  # Node 1
    [1, 0, 1, 1, 1],  # Node 2
    [0, 1, 0, 0, 0],  # Node 3
    [1, 1, 0, 0, 1],  # Node 4
    [0, 1, 1, 1, 0]   # Node 5
], dtype=float)


N = adj_matrix.shape[0]

pagerank = np.full(N, 1/N)


outbound_link_count = np.sum(adj_matrix, axis=0)


threshold = 1e-6


change = 1
iterations = 0

while change > threshold:
    new_pagerank = np.zeros(N)

    for i in range(N):
        for j in range(N):
            if adj_matrix[j][i] == 1:
                new_pagerank[i] += pagerank[j] / outbound_link_count[j]
        new_pagerank[i] = (1 - d) / N + d * new_pagerank[i]

    change = np.sum(np.abs(new_pagerank - pagerank))
    pagerank = new_pagerank
    iterations += 1

print("Nilai PageRank setelah konvergensi:")
for i in range(N):
    print(f"Node {i+1}: {pagerank[i]:.4f}")

print(f"Jumlah iterasi: {iterations}")

Node 3: 0.2315
Node 4: 0.2315
Node 5: 0.1606
Jumlah iterasi: 17

penjelasan code tersebut

1.Definisikan faktor redaman: Set faktor redaman d=0.85.

2.Definisikan matriks adjacency: Matriks adjacency sesuai dengan graf yang diberikan.

3.Inisialisasi nilai PageRank: Setiap matriks memulai dengan PageRank sebesar 1/5 karena ada 5 node.

4.Hitung jumlah tautan keluar: Jumlah tautan keluar untuk setiap matriks

5.Tentukan ambang batas konvergensi: Threshold untuk menentukan kapan iterasi berhenti.

6.Perhitungan PageRank: Iterasi perhitungan PageRank hingga perubahan antara iterasi kurang dari threshold.

7.Output: Cetak nilai PageRank untuk setiap node dan jumlah iterasi yang diperlukan untuk konvergensi.

2.Carilah penyelesaian dari Sistem persamaan berikut dengan Eliminasi Gauss

cara untuk mencari sistem persamaan menggunakan eliminasi gaus tersebut kita harus cari langkah langkah nya terlebih dahulu agar gampang di mengerti

1.tulis sitem persamaan dalam bentuk matriks augmented

2.gunakan eliminasi gauss untuk mengubah matriks augmented ke dalam bentuk eselon baris

3.Kurangi baris kedua dengan 2 kali baris pertama, dan baris ketiga dengan 6 kali baris pertama

4.Bagi baris kedua dengan 3, dan kurangi baris ketiga dengan 6 kali baris kedua yang telah dibagi:

5.Bagi baris ketiga dengan -9

6.Eliminasi elemen di atas dan di bawah pivot terakhir

7.Eliminasi elemen di atas pivot kedua

import numpy as np

A = np.array([
    [1, -1, 1, 3],
    [2, 1, 3, 8],
    [6, 0, -1, 4]
], dtype=float)

def gauss_elimination(A):
    rows, cols = A.shape

    for i in range(rows):
        # Pivoting
        A[i] = A[i] / A[i, i]
        for j in range(i + 1, rows):
            A[j] = A[j] - A[j, i] * A[i]


    for i in range(rows - 1, -1, -1):
        for j in range(i - 1, -1, -1):
            A[j] = A[j] - A[j, i] * A[i]

    return A[:, -1]


solution = gauss_elimination(A.copy())
print("penyelesaian sistem dari persamaan berikut adalah:")
for i, sol in enumerate(solution, start=1):
    print(f"x_{i} = {sol:.4f}")

penyelesaian sistem dari persamaan berikut adalah:
x_1 = 1.0000
x_2 = 0.0000
x_3 = 2.0000

penjelasan tentang code tersebut

1.Matriks augmented A dibentuk dari sistem persamaan yang diberikan.

2.Fungsi gauss_elimination: Fungsi ini melakukan eliminasi Gauss untuk mengubah matriks augmented menjadi bentuk eselon baris tereduksi.Eliminasi ke bentuk eselon baris: Untuk setiap baris, elemen pivot dinormalisasi dan kemudian digunakan untuk mengeliminasi elemen di bawahnya. Back substitution: Setelah mencapai bentuk eselon baris, back substitution digunakan untuk mengeliminasi elemen di atas pivot, menghasilkan bentuk eselon baris tereduksi.

3.Penerapan eliminasi Gauss: Fungsi gauss_elimination diterapkan pada matriks augmented untuk menemukan solusi sistem persamaan.

4.output : solusi sitem persamaan di cetak
